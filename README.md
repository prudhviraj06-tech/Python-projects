# 🏧 ATM Management System (Python)

## 📌 Project Overview

This is a simple **ATM Management System** developed using **Python**. The project simulates the basic functionalities of an ATM through a command-line interface (CLI). It allows users to securely log in using a PIN and perform common banking operations such as checking their balance, depositing money, withdrawing money, and viewing recent transactions.

This project is suitable for Python beginners to understand programming concepts like loops, conditional statements, lists, variables, and user input.

---

## ✨ Features

* 🔐 Secure PIN authentication
* 🚫 Maximum 3 login attempts
* 💰 Balance inquiry
* ➕ Deposit money
* ➖ Withdraw money
* 📜 View the last 5 transactions
* 🚪 Exit the ATM system safely
* ✅ Input validation for deposits and withdrawals

---


3. Run the Python program:
